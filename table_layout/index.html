Absolute vs Relative Links

Absolute link: An absolute link is a link that includes the entire (e.g. "https://www.google.com") . Absolute links are used to link to other websites to more information/resources.

Relative Link: A relative link is a link that only includes a portion of the URL, typically just the path and filename of the resource being linked to. Relative links are used to link to resources on the same domain related to the current page.

It is important to understand the difference between relative and absolute links because it affects how links are interpreted by web browsers and search engines. As a Web Developer, it is important to know these things.
